# DiceGame

## Stage 1

### Requirements:

一个玩家，掷骰子两次，和为 7 则获胜，否则失败。

要求控制台介绍游戏规则，并提示操作方式，如Please input your command('s' to start, 'r' to roll the dice ,'q' to quit)。

要求打印每次操作后的提示、每次掷骰子的结果，和玩家的胜负
